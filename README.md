Hugging Face Model Explorer

##Introduction

This project is developed as part of the AFE (Advanced Front-End Engineering) course. The main objective of this project is to study and present information about a trending AI model available on the Hugging Face platform. The website provides a structured overview of the selected model, including its architecture, dataset, training details, capabilities, advantages, limitations, and practical applications.

Artificial Intelligence has become one of the fastest-growing technologies in the world, and Hugging Face has become one of the most popular platforms for sharing and deploying machine learning models. This project helps users understand these concepts through a simple and interactive web interface.



#  Objectives

The objectives of this project are:

- To understand the Hugging Face ecosystem.
- To explore trending AI and Machine Learning models.
- To study how pre-trained models are developed and shared.
- To understand datasets used for model training.
- To learn about model architecture and training parameters.
- To create an informative website using HTML, CSS, and JavaScript.
- To improve front-end development skills by designing a responsive webpage.
#  About Hugging Face

Hugging Face is an AI company that provides open-source machine learning models, datasets, and tools for developers, researchers, and students. It has become one of the largest communities for Artificial Intelligence.

The Hugging Face Hub contains thousands of models that can perform tasks such as:

- Text Generation
- Language Translation
- Text Summarization
- Image Classification
- Object Detection
- Speech Recognition
- Question Answering
- Sentiment Analysis
- Chatbot Development

Developers can download these models or use them directly through APIs.

#2   LIVE NEWS

Project Overview
The Live News Website is a responsive web application that provides users with the latest news from various categories, including Technology, Business, Sports, Entertainment, Health, Science, and General. The application retrieves real-time news data using a News API and displays it in a simple, organized, and user-friendly interface.


Features
Displays the latest news articles in real time.
Search news articles using keywords.
Browse news by category.
Responsive design for desktop, tablet, and mobile devices.
Displays news title, image, description, source, and publication date.
Provides a "Read More" link to view the complete article on the original news website.
Fast loading and easy navigation.


Technologies Used
HTML5
CSS3
JavaScript (ES6)
News API
Fetch API



Project Structure
index.html – Main webpage structure.
style.css – Styles and layout of the website.
script.js – Fetches news data from the API and updates the webpage dynamically.
images/ – Stores icons and other image assets.


How It Works
The application sends a request to the News API.
The API returns the latest news articles in JSON format.
JavaScript processes the received data.
News articles are displayed dynamically as cards containing the headline, image, description, source, and publication date.
Users can search for specific topics or browse different news categories.


Learning Outcomes
This project helped in understanding:
API integration using JavaScript.
Fetching and handling JSON data.
Dynamic DOM manipulation.
Responsive web design.
Error handling during API requests.
Building a real-time web application.


Future Enhancements
